## 插件命令
1、安装DocPad模块
<blockquote>
npm install -g npm; npm install -g docpad
</blockquote>

2、新建项目
<blockquote>
docpad update,init
</blockquote>

3、安装DocPad插件
<blockquote>
docpad install eco,ghpages,livereload,marked
</blockquote>

4、发布GitHub页面
<blockquote>
docpad deploy-ghpages --env static
</blockquote>
